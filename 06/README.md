# 6章 アセンブラ

## 使用法

```$ ./Assembler.py <asm ファイル>```

## 実装

- C 命令の判定には正規表現ではなく split() を使用して条件分岐で処理した
- ```for line in f.readline():``` のようなコードは書かない
  - 書くなら ```for line in f:```
- SymbolTable クラスの初期化ではテーブルは空とテキストに記載されていたが定義済みシンボルだけ追加するようにした
- ```M=D```（命令）や ```// comment```（コメントアウト）のパターンだけでなく、```M=D   // comment``` のような混合パターンもある点に注意
- コンパイラが asm ファイルを正しく出力してくれる（はずである）ので、正規表現はある程度大雑把でもよい
