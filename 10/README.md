## 第1段階：トークナイザ

### 方針

1. 1行読んでコメントもしくは API コメントの途中なら何もせずに次の行へ
2. 読み込んだソースコードの1行からトークンリストを作成する
   - 例：```if (x < 153)``` → ```['if', '(', 'x', '&lt;', '153', ')']```
3. トークンリストを pop し各要素がどのトークンタイプかを見て xml ファイルを記述していく
